# Ruby on Rails - Car-reviews-rails-application
> This is a website for a care review company in kenya

## Built With

- Ruby 2.5.5
- Ruby on rails 6.0.0
- VS Code
- SQLite database

## How to Install

- Install Ruby in your system. To install Ruby, you can read the [Documentation](https://www.ruby-lang.org/en/documentation/installation/)
- Install rails to do that run this command in your console
    ``` 
    gem install rails

    ``` 
- Then For Windows users, open Command Prompt and run:
    ```console
    C:\>cd Car-reviews-rails-application
    C:\cd Car-reviews-rails-application> run rails server
    ```
- Open another terminal or Command line and run:  

  ```
    $ rails db:migrate
    
    ``` 
- For Linux and Mac users, open a Terminal and run:
    ```console
    
    $ cd Car-reviews-rails-application folder
    $ run rails server

    ``` 
- Navigate to your browser and run
 
   ```
   localhost:3000

   ```
